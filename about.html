About

Phone Compare Pro is a lightweight, responsive tool to help you shortlist and compare smartphones quickly. It uses a client‑side dataset and modern UI patterns so it runs fast even on low bandwidth.

  • Responsive grid and sticky filters
  • Dark mode, mobile menu, and smooth reveal animations
  • Two‑item compare with “better” highlights
  • Favorites powered by localStorage
